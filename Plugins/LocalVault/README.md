# LocalVault Core

LocalVault Core is the Open Source version of the LocalVault plugin for Unreal Engine. It allows you to synchronize your Epic Fab asset cache and visually organize them with a custom tagging system directly inside the Editor.

## Features (Core)
- **Fab Cache Sync:** Automatically reads your Epic Fab cache and lists all downloaded assets.
- **Custom Tagging:** Apply custom tags (e.g., "Sci-Fi", "Weapons", "Foliage") to any asset.
- **Filtering & Search:** Instantly filter your library by Author, Title, or Tag.

## LocalVault Pro
Upgrade to LocalVault Pro on Fab to get the ultimate offline storage solution:
- **True Offline Vault:** Securely backup assets out of the temporary cache into a dedicated vault directory.
- **Bulk Downloading:** Download multiple assets simultaneously directly from the UI.
- **Auto-Updates:** Automatically check for updates on all your offline assets.
- **Custom Storage Paths:** Define where your assets are stored per-project or globally.
